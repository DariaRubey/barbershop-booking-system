Как мастер, я хочу просматривать записи на услуги клиентов и отзывы клиента о выполненной работе

| **Исключение** | 1Е. Мастер открыл раздел с расписанием1.Система сообщает, что записей нет. |
| **Приоритет** | Средний |

UC-5 Посмотреть отзывы  на выполненные услуги 

| Раздел | Содержание |
| ----- | ----- |
| **Идентификатор и название** | UC-4 Посмотреть отзывы  на выполненные услуги |
| **Автор** | Дарья |
| **Дата создания** | 15.06.26 |
| **Основное действующее лицо** | Мастер |
| **Роль действующего лица** | Сотрудник, который хочет посмотреть отзывы |
| **Потребность** | Записаться на услугу онлайн |
| **Бизнес-цель** | Хочет видеть оценки и отзывы клиентов, чтобы понимать качество своей работы  |
| **Дополнительные действующие лица** | КлиентМенеджер |
| **Тип связи** | Основная UC |
| **Описание** | Мастер просматривает отзывы и оценки клиентов по выполненным им услугам. Система показывает только отзывы, относящиеся к работам выбранного мастера.  |
| **Триггер** | Мастер открыл раздел с расписанием |
| **Предварительные условия** | Мастер авторизован в системе. Мастер имеет право просматривать отзывы по своим работам  В системе есть выполненные услуги мастера. Клиенты оставили оценки или отзывы после получения услуги.  |
| **Выходные условия** | Минимальные гарантии: Система не показывает отзывы на других мастеров Гарантии успеха: Мастер получил список отзывов по своим выполненным работам. Мастер видит оценку, текст отзыва, дату услуги и связанную услугу   |
| **Нормальное направление развития варианта использования** | Мастер открыл просмотр отзывов клиентов Мастер выбирает период, услугу или другой доступный параметр просмотра Система получает отзывы клиентов по выполненным работам мастера Система показывает мастеру список отзывов с оценкой, текстом отзыва, датой и услугой Мастер просматривает отзывы клиентов  |
| **Исключение** | 2.E Мастер выбирает период, услугу или другой доступный параметр просмотра1\. Система сообщает мастеру, что данные отзывов временно недоступны.2\. Система не показывает неполные или неподтвержденные данные.3\. UC завершается без просмотра отзывов |
| **Приоритет** | Средний |
