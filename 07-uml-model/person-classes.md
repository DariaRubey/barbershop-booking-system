Класс Персоны  
**Назначение класса:** Класс, пердназнаенный для хранения общих данных о персонах (людях) взаимодействующих системой онлайн-записи барбершопа. Класс содержит регистрационные и контактные данные человека. Используется в процессах регистрации, бронирования, составления расписания, для определения ролей и выдачи прав доступа.  
**Атрибуты**

| Ключ | Наименование атрибута | Мнемоника | Описание атрибута | Тип данных | Обязательность | Примечание |
| :---- | :---- | :---- | :---- | :---- | :---- | :---- |
| PK | Идентификатор персоны  | person\_id  | Уникальный идентификатор записи о человеке в системе  | integer  | Да, генерируется системой | Используется для связи с сущностями “Клиенты” и “Сотрудники”  |
|  | Логин | login  | Регистрационные данные для входа в систему  | varchar (50)  | Да | Может формироваться системой или задаваться при регистрации  |
|  | Пароль | password\_hash  | Пароль для доступа в систему  | varchar (255) | Да | При регистрации клиента менеджером система генерирует временный пароль  |
|  | Имя | full\_name | Используется для обращений в СМС, уведомлениях, интерфейсе | varchar (100) | Да | Формат: "Имя" или "Имя Отчество" |
|  | Номер телефона | phone\_number | используется для регистрации, дозвонов и отправки СМС | varchar (20) | Да | Уникальный. Формат международный (например, \+7XXXXXXXXXX) |
|  | Дата регистрации  | registration\_date  | Датасоздания записи о персоне в системе  | date | Да  | Заполняется автоматически при регистрации.  |
