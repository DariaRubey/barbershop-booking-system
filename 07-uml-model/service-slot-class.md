Сущность Слоты обслуживания для расписания  
**Назначение сущности:** Сущность предназначена для хранения временных интервалов, в которые конкретный мастер может принять клиента.  
ую услугу  
Сущность используется менеджером при формировании расписания мастера и системой при отображении свободных слотов клиенту 

Один экземпляр сущности **«Слоты обслуживания»** соответствует одному временному интервалу работы конкретного мастера.  
В этой сущности **не храним клиента**, потому что слот может существовать до бронирования.

Сущность Бронирование слота (Запись на услугу)  
**Назначение сущности:** сущность, предназначенная для хранения факта бронирования клиентом конкретного слота обслуживания.  
Сущность связывает клиента, выбранный слот и выбранную услугу. Она создаётся после того, как клиент выбрал свободный слот, услугу, мастера и подтвердил бронирование.  
Один экземпляр сущности «Запись на услугу» соответствует одной записи клиента на обслуживание.
